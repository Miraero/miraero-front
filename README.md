# Miraero-front

테오의 구글 스프린트 12기 - 미래편지팀의 프로젝트 프론트엔드 저장소입니다!

## 구동법

### Clone

`git clone https://github.com/miraero/miraero-front`

### Setup

`npm install` 또는 `yarn`

### Run

`npm run dev` 또는 `yarn run dev`

### Build

`npm run build`

## Environment configuration

```
SERVER_URI
```

## 사용한 기술

-   ReactJS
    -   Emotion.js / Styled-components
    -   Next.js : SSR,Page Routes
    -   React Query : 비동기 통신
